## **Description**
We are going to build an OwlExpress-like application, where students will register for courses, staff can update course offering, and where faculty can input the course grade of students.

## **Requirements**
- Students can register to multiple courses.

- Staff members can create, update, and delete courses in the system.

- Faculty members can input the course grade of a student registered in their classes

## **Rules and relationships**
- The time of the courses a student registers should not conflict.

- Students will have one course grade for each course they register for.

- Faculty members will be assigned to courses.

- A faculty member can be assigned to multiple courses.

- The time of the courses a faculty member is assigned should not conflict.

## **User stories**
- As a **student**, I want to be able to **sign in to the system** so I can securely access my academic information.

- As a **student**, I want to be able to **browse the courses being offered next semester**, so I can plan my schedule in advance.

- As a **student**, I want to be able to **register for courses**, so I can build my academic schedule.

- As a **student**, I want to be able to **view my schedule**, so I can organize and manage my time.

- As a **student**, I want to be able to **view my grades**, so that I can track my academic performance.

- As a **staff member**, I want to be able to **sign in to the system**, so that I can manage course offerings securely.

- As a **staff member**, I want to be able to **add new courses to the system**, so that students and faculty can see and register for them.

- As a **staff member**, I want to be able to **update existing course details**, so that the course catalog remains accurante and current.

- As a **staff member**, I want to be able to **removew outdated or canceled courses**, so that students do not register for unavailable courses.

- As a **faculty member**, I want to be able to **sign in to the system**, so that I can access and manage grades.

- As a **faculty member**, I want to be able to **enter or update student final course grade**, so that students can see how they did in the course at the end of the semester.
